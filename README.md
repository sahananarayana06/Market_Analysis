# 📊 Market Analysis using Python

## 📌 Project Overview
This project analyzes a retail sales dataset using Python. It includes data cleaning, exploratory data analysis (EDA), data visualization, and a machine learning model to predict sales.

The objective is to identify sales trends, analyze business performance, and build a basic sales prediction model using Linear Regression.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- VS Code

---

## 📈 Exploratory Data Analysis

The following analyses were performed:

- Checked dataset structure and data types
- Identified missing values
- Removed duplicate records
- Calculated total sales and total profit
- Found top-selling product sub-categories
- Analyzed sales distribution across categories
- Compared profit across regions

---

## 📊 Visualizations

The project includes:
- Top 10 Sub-Categories by Sales (Bar Chart)
- Sales Distribution by Category (Pie Chart)
- Profit by Region (Bar Chart)
- Quantity vs Sales (Scatter Plot)
